## 📌 About

안녕하세요.

이 사이트는 **1차 캐릭터 CP ORPG 기록을 백업하기 위해 제작된 개인 사이트**입니다.
특별한 목적보다는 **"꾸미고 싶다"는 단순한 욕망 하나로 GitHub Pages를 이용해 구축**하게 되었습니다.

## 🎯 제작 의도

* **ORPG 기록 보관 및 백업**
* 캐릭터 및 스토리 관련 자료 정리
* 개인적인 창작 아카이브

해당 사이트는 **개인 창작 활동을 위한 공간**이며
다른 의도는 일절 없음을 밝힙니다.

## ⚠️ 안내

GitHub는 **공개 저장소**이기 때문에
일부 HTML 파일은 **간단한 보호 처리를 해두었지만 완전하지 않을 수 있습니다.**

따라서 **스토리 스포일러가 발생할 수 있으니 열람은 가급적 자제해 주시면 감사하겠습니다.**
